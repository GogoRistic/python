# Projekttitel

**Projekt python**

## Beschreibung

Dieses Projekt ist ein Python-basiertes Wort-Scramble-Spiel, bei dem der Spieler zufällig gemischte Wörter richtig erraten muss.
Der Nutzer erhält 60 Sekunden pro Wort und maximal 3 Versuche.
Das Spiel umfasst 5 Runden, ein Punktesystem und eine einfache Rangliste.

## Installation

**Voraussetzungen**

Python 3.10 oder höher

Keine externen Libraries notwendig (nur random und time, beide Bestandteil der Standardbibliothek)

Installation und Start

Projekt herunterladen oder Repository klonen

Datei wort_scramble.py starten:

python wort_scramble.py

## Verwendung

Nach dem Start des Programms:

Nutzername eingeben

In jeder Runde das gemischte Wort versuchen zu lösen

Punkte entsprechend der Leistung sammeln

Nach 5 Runden erscheint die Gesamtauswertung und Rangliste

**Beispielausführung**
Wie heisst du? Gogo

Willkommen zum Wort-Scramble Spiel, Gogo!
Du hast 60 Sekunden pro Wort und 3 Versuche.

Runde 1
Dein Wort: ekzta
Deine Antwort: Katze
Richtig!
Aktueller Punktestand: 6
------------------------------

## Features

- Zufallswoerter aus einer umfangreichen deutschen Wortliste

- Automatisches Scrambling der Buchstaben

- Zeitlimit pro Runde (60 Sekunden)

- Punktesystem basierend auf der Anzahl der Versuche

- 5-Runden-Spielstruktur

- Dynamische Rangliste

- Schutz vor Wortwiederholungen in einer Sitzung

## Lizenz

Dieses Projekt kann frei verwendet und erweitert werden.
